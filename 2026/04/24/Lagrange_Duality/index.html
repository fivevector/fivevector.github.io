统计学习方法 Notes: Lagrange Duality

Lagrange Duality is an useful tool which could be used in SVM(Machine Learning) and some algorithms.

In constrained optimization, Lagrange duality is frequently used to transform a primal problem into a dual problem. By solving the dual, we can often find the solution to the primal more easily.

The Primal Problem

Consider a constrained optimization problem where f(x)f(x)f(x), ci(x)c_i(x)ci​(x), and hj(x)h_j(x)hj​(x) are continuous, differentiable functions defined on Rn\mathbb{R}^nRn.

The primal optimization problem is defined as:

min⁡x∈Rnf(x)\min_{x \in \mathbb{R}^n} f(x) x∈Rnmin​f(x)

s.t.ci(x)≤0,i=1,2,…,ks.t. \quad c_i(x) \le 0, \quad i = 1, 2, \dots, k s.t.ci​(x)≤0,i=1,2,…,k

hj(x)=0,j=1,2,…,lh_j(x) = 0, \quad j = 1, 2, \dots, l hj​(x)=0,j=1,2,…,l

To solve this, we first introduce the generalized Lagrange function:

L(x,α,β)=f(x)+∑i=1kαici(x)+∑j=1lβjhj(x)L(x, \alpha, \beta) = f(x) + \sum_{i=1}^k \alpha_i c_i(x) + \sum_{j=1}^l \beta_j h_j(x) L(x,α,β)=f(x)+i=1∑k​αi​ci​(x)+j=1∑l​βj​hj​(x)

where:

  • x=(x(1),x(2),…,x(n))T∈Rnx = (x^{(1)}, x^{(2)}, \dots, x^{(n)})^T \in \mathbb{R}^n x=(x(1),x(2),…,x(n))T∈Rn

  • Lagrange Multipliers are αi\alpha_iαi​ and βj\beta_jβj​, where we strictly require αi≥0\alpha_i \ge 0αi​≥0 for the inequality constraints.

Next, we define a function of xxx called the primal objective function, denoted as θP(x)\theta_P(x)θP​(x):

θP(x)=max⁡α,β:αi≥0L(x,α,β)\theta_P(x) = \max_{\alpha, \beta: \alpha_i \ge 0} L(x, \alpha, \beta) θP​(x)=α,β:αi​≥0max​L(x,α,β)

Then we could give a brief analysis on the θP(x)\theta_P(x)θP​(x):

  • If xxx violates any constraint, we can let αi→+∞\alpha_i \to +\inftyαi​→+∞ or βj\beta_jβj​ be a very large value (positive or negative) such that βjhj(x)→+∞\beta_j h_j(x) \to +\inftyβj​hj​(x)→+∞. In all these cases θP(x)=+∞\theta_P(x) = +\inftyθP​(x)=+∞.

  • If xxx satisfies all constraints, since ci(x)≤0c_i(x) \le 0ci​(x)≤0 and αi≥0\alpha_i \ge 0αi​≥0, the term αici(x)\alpha_i c_i(x)αi​ci​(x) is at most 000. To maximize the sum, we set αi=0\alpha_i = 0αi​=0 (if ci(x)<0c_i(x) < 0ci​(x)<0) or it remains 000 (if ci(x)=0c_i(x) = 0ci​(x)=0).In this case, θP(x)=f(x)\theta_P(x) = f(x)θP​(x)=f(x).

Thus, the function is simplified to:

θP(x)={f(x),if x satisfies all primal constraints+∞,otherwise\theta_P(x) = \begin{cases} f(x), & \text{if } x \text{ satisfies all primal constraints} \\+\infty, & \text{otherwise} \end{cases} θP​(x)={f(x),+∞,​if x satisfies all primal constraintsotherwise​

Therefore, the problem min⁡xθP(x)\min_x \theta_P(x)minx​θP​(x) is equivalent to the original primal problem. We define the optimal value of the primal problem as:

p∗=min⁡xθP(x)=min⁡xmax⁡α,β:αi≥0L(x,α,β)p^* = \min_x \theta_P(x) = \min_x \max_{\alpha, \beta: \alpha_i \ge 0} L(x, \alpha, \beta) p∗=xmin​θP​(x)=xmin​α,β:αi​≥0max​L(x,α,β)

The Dual Problem

The dual problem is created by flipping the order of the “min” and “max” operations.

First, we define the dual objective function:

θD(α,β)=min⁡xL(x,α,β)\theta_D(\alpha, \beta) = \min_x L(x, \alpha, \beta) θD​(α,β)=xmin​L(x,α,β)

Then, we seek to maximize this function with respect to the multipliers α\alphaα and β\betaβ (subject to αi≥0\alpha_i \ge 0αi​≥0):

max⁡α,β:αi≥0θD(α,β)=max⁡α,β:αi≥0min⁡xL(x,α,β)\max_{\alpha, \beta: \alpha_i \ge 0} \theta_D(\alpha, \beta) = \max_{\alpha, \beta: \alpha_i \ge 0} \min_x L(x, \alpha, \beta) α,β:αi​≥0max​θD​(α,β)=α,β:αi​≥0max​xmin​L(x,α,β)

This is known as the max-min problem. We define the optimal value of the dual problem as d∗d^*d∗:

d∗=max⁡α,β:αi≥0θD(α,β)d^* = \max_{\alpha, \beta: \alpha_i \ge 0} \theta_D(\alpha, \beta) d∗=α,β:αi​≥0max​θD​(α,β)

Relationship Between Primal and Dual Problems

Theorem 1 Weak Duality:
If both the primal and dual problems have optimal values, then the dual optimal value d∗d^*d∗ is always less than or equal to the primal optimal value p∗p^*p∗:

d∗=max⁡α,β:αi≥0min⁡xL(x,α,β)≤min⁡xmax⁡α,β:αi≥0L(x,α,β)=p∗d^* = \max_{\alpha, \beta: \alpha_i \ge 0} \min_x L(x, \alpha, \beta) \le \min_x \max_{\alpha, \beta: \alpha_i \ge 0} L(x, \alpha, \beta) = p^* d∗=α,β:αi​≥0max​xmin​L(x,α,β)≤xmin​α,β:αi​≥0max​L(x,α,β)=p∗

Proof:∀x,α,β\forall x, \alpha, \beta∀x,α,β, the following inequality holds by definition:

θD(α,β)=min⁡xL(x,α,β)≤L(x,α,β)≤max⁡α,β:αi≥0L(x,α,β)=θP(x)\theta_D(\alpha, \beta) = \min_x L(x, \alpha, \beta) \le L(x, \alpha, \beta) \le \max_{\alpha, \beta: \alpha_i \ge 0} L(x, \alpha, \beta) = \theta_P(x) θD​(α,β)=xmin​L(x,α,β)≤L(x,α,β)≤α,β:αi​≥0max​L(x,α,β)=θP​(x)

Since θD(α,β)≤θP(x)\theta_D(\alpha, \beta) \le \theta_P(x)θD​(α,β)≤θP​(x) for all variables, it follows that the maximum of the lower bound (θD\theta_DθD​) cannot exceed the minimum of the upper bound (θP\theta_PθP​).

Theorem 2 Strong Duality:
If f(x)f(x)f(x) and ci(x)c_i(x)ci​(x) are convex functions, hj(x)h_j(x)hj​(x) are affine functions (hj(x)=ajTx+bjh_j(x) = a_j^T x + b_jhj​(x)=ajT​x+bj​), and the inequality constraints are strictly feasible (Slater’s condition: ∃x\exists x∃x such that ci(x)<0c_i(x) < 0ci​(x)<0 for all iii), then there exist x∗,α∗,β∗x^*, \alpha^*, \beta^*x∗,α∗,β∗ such that x∗x^*x∗ is the primal solution, α∗,β∗\alpha^*, \beta^*α∗,β∗ are the dual solutions, and:

p∗=d∗=L(x∗,α∗,β∗)p^* = d^* = L(x^*, \alpha^*, \beta^*) p∗=d∗=L(x∗,α∗,β∗)

Proof:

Lemma 1 Hyperplane Separation Theorem:
Let CCC and DDD be two non-empty disjoint convex sets in Rn\mathbb{R}^nRn (i.e., C∩D=∅C \cap D = \emptysetC∩D=∅). Then there exists a non-zero vector a∈Rna \in \mathbb{R}^na∈Rn and a scalar b∈Rb \in \mathbb{R}b∈R such that:

aTx≤bfor all x∈Ca^T x \le b \quad \text{for all } x \in C aTx≤bfor all x∈C

and

aTy≥bfor all y∈Da^T y \ge b \quad \text{for all } y \in D aTy≥bfor all y∈D

The equation aTx=ba^T x = baTx=b defines a hyperplane that separates the two sets. The vector aaa is the normal vector to the hyperplane.

Lemma 2 Supporting Hyperplane Theorem:

Let CCC be a non-empty convex set in Rn\mathbb{R}^nRn, and let x0x_0x0​ be a point on the boundary of CCC (i.e., x0∈∂Cx_0 \in \partial Cx0​∈∂C). Then there exists a non-zero vector a∈Rna \in \mathbb{R}^na∈Rn such that:

aTx≤aTx0for all x∈Ca^T x \le a^T x_0 \quad \text{for all } x \in C aTx≤aTx0​for all x∈C

Let G={(c1(x),…,ck(x),h1(x),…,hl(x),f(x))∈Rk+l+1∣x∈Rn}\mathcal{G} = \{ (c_1(x), \dots, c_k(x), h_1(x), \dots, h_l(x), f(x)) \in \mathbb{R}^{k+l+1} \mid x \in \mathbb{R}^n \}G={(c1​(x),…,ck​(x),h1​(x),…,hl​(x),f(x))∈Rk+l+1∣x∈Rn}.

Because fff and cic_ici​ are convex and hjh_jhj​ is affine, we can define a convex set A\mathcal{A}A that “sits above” G\mathcal{G}G.

According to the lemma 2, there exists a hyperplane that separates the set of unattainable values (values below p∗p^*p∗) from the set of attainable values A\mathcal{A}A. And this hyperplane defines the optimal Lagrange multipliers α∗\alpha^*α∗ and β∗\beta^*β∗.

Slater’s condition ensures that the hyperplane is not “vertical” (i.e., the coefficient associated with f(x)f(x)f(x) is non-zero).

The equation of this hyperplane in the (u,v,t)(u, v, t)(u,v,t) space is:

t+∑i=1kαi∗ui+∑j=1lβj∗vj≥ct + \sum_{i=1}^k \alpha_i^* u_i + \sum_{j=1}^l \beta_j^* v_j \ge c t+i=1∑k​αi∗​ui​+j=1∑l​βj∗​vj​≥c

Since the hyperplane passes through the boundary point (0,0,p∗)(0, 0, p^*)(0,0,p∗), we substitute these coordinates to find the constant ccc:

p∗+∑αi∗(0)+∑βj∗(0)=c  ⟹  c=p∗p^* + \sum \alpha_i^*(0) + \sum \beta_j^*(0) = c \implies c = p^* p∗+∑αi∗​(0)+∑βj∗​(0)=c⟹c=p∗

Thus, for any point (u,v,t)∈A(u, v, t) \in \mathcal{A}(u,v,t)∈A, the following inequality holds:

t+∑i=1kαi∗ui+∑j=1lβj∗vj≥p∗t + \sum_{i=1}^k \alpha_i^* u_i + \sum_{j=1}^l \beta_j^* v_j \ge p^* t+i=1∑k​αi∗​ui​+j=1∑l​βj∗​vj​≥p∗

By the definition of set A\mathcal{A}A, for any x∈Rnx \in \mathbb{R}^nx∈Rn, the point (c(x),h(x),f(x))(c(x), h(x), f(x))(c(x),h(x),f(x)) is an element of A\mathcal{A}A.

Substituting these into the hyperplane equation gives us:

f(x)+∑i=1kαi∗ci(x)+∑j=1lβj∗hj(x)≥p∗f(x) + \sum_{i=1}^k \alpha_i^* c_i(x) + \sum_{j=1}^l \beta_j^* h_j(x) \ge p^* f(x)+i=1∑k​αi∗​ci​(x)+j=1∑l​βj∗​hj​(x)≥p∗

The left-hand side is exactly the Generalized Lagrangian Function L(x,α∗,β∗)L(x, \alpha^*, \beta^*)L(x,α∗,β∗). Therefore:

L(x,α∗,β∗)≥p∗∀x∈RnL(x, \alpha^*, \beta^*) \ge p^* \quad \forall x \in \mathbb{R}^n L(x,α∗,β∗)≥p∗∀x∈Rn

The dual function θD(α,β)\theta_D(\alpha, \beta)θD​(α,β) is defined as the infimum of the Lagrangian over xxx. Since the inequality above holds for all xxx, it must also hold for the minimum:

θD(α∗,β∗)=inf⁡xL(x,α∗,β∗)≥p∗\theta_D(\alpha^*, \beta^*) = \inf_{x} L(x, \alpha^*, \beta^*) \ge p^* θD​(α∗,β∗)=xinf​L(x,α∗,β∗)≥p∗

By definition, the optimal dual value d∗d^*d∗ is the maximum of θD(α,β)\theta_D(\alpha, \beta)θD​(α,β). Since we have found a specific pair (α∗,β∗)(\alpha^*, \beta^*)(α∗,β∗) that yields a value ≥p∗\ge p^*≥p∗, we conclude:

d∗=max⁡α,βθD(α,β)≥θD(α∗,β∗)≥p∗d^* = \max_{\alpha, \beta} \theta_D(\alpha, \beta) \ge \theta_D(\alpha^*, \beta^*) \ge p^* d∗=α,βmax​θD​(α,β)≥θD​(α∗,β∗)≥p∗

Which gives us:

We also have the Weak Duality property, then:

d∗=p∗d^* = p^* d∗=p∗

Theorem 3 KKT Conditions:
Given the same convexity and feasibility assumptions as Theorem 2, the points x∗,α∗,β∗x^*, \alpha^*, \beta^*x∗,α∗,β∗ are the optimal solutions to the primal and dual problems if and only if they satisfy the Karush-Kuhn-Tucker (KKT) conditions:

  • ∇xL(x∗,α∗,β∗)=0\nabla_x L(x^*, \alpha^*, \beta^*) = 0∇x​L(x∗,α∗,β∗)=0 (Stationarity)

  • ci(x∗)≤0,i=1,…,kc_i(x^*) \le 0, \quad i=1, \dots, kci​(x∗)≤0,i=1,…,k (Primal Feasibility)

  • αi∗≥0,i=1,…,k\alpha_i^* \ge 0, \quad i=1, \dots, kαi∗​≥0,i=1,…,k (Dual Feasibility)

  • hj(x∗)=0,j=1,…,lh_j(x^*) = 0, \quad j=1, \dots, lhj​(x∗)=0,j=1,…,l (Primal Feasibility)

Proof:

Since x∗x^*x∗ minimizes L(x,α∗,β∗)L(x, \alpha^*, \beta^*)L(x,α∗,β∗) (from the definition of the dual), its gradient must be zero: ∇xL=0\nabla_x L = 0∇x​L=0.

We know p∗=d∗p^* = d^*p∗=d∗, which implies:

f(x∗)≤f(x∗)+∑αi∗ci(x∗)+∑βj∗hj(x∗)f(x^*) \le f(x^*) + \sum \alpha_i^* c_i(x^*) + \sum \beta_j^* h_j(x^*) f(x∗)≤f(x∗)+∑αi∗​ci​(x∗)+∑βj∗​hj​(x∗)

Since hj(x∗)=0h_j(x^*) = 0hj​(x∗)=0 and αi∗≥0,ci(x∗)≤0\alpha_i^* \ge 0, c_i(x^*) \le 0αi∗​≥0,ci​(x∗)≤0, the summation ∑αi∗ci(x∗)\sum \alpha_i^* c_i(x^*)∑αi∗​ci​(x∗) must be ≤0\le 0≤0.

For the inequality f(x∗)≤f(x∗)+non-positive termf(x^*) \le f(x^*) + \text{non-positive term}f(x∗)≤f(x∗)+non-positive term to hold as an equality, we must have ∑αi∗ci(x∗)=0\sum \alpha_i^* c_i(x^*) = 0∑αi∗​ci​(x∗)=0. Since each term is ≤0\le 0≤0, each individual term αi∗ci(x∗)\alpha_i^* c_i(x^*)αi∗​ci​(x∗) must be zero.

Sufficiency:

From KKT-1, x∗x^*x∗ is a global minimum of L(x,α∗,β∗)L(x, \alpha^*, \beta^*)L(x,α∗,β∗) because LLL is convex in xxx. Therefore, θD(α∗,β∗)=L(x∗,α∗,β∗)\theta_D(\alpha^*, \beta^*) = L(x^*, \alpha^*, \beta^*)θD​(α∗,β∗)=L(x∗,α∗,β∗).

Using KKT-2 (slackness) and KKT-5 (equality constraints):
