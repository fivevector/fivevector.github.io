Data Structure2:Balanced Binary Search Tree

Binary Search Tree

二叉搜索树是一种查找、插入、删除在平均情况下均为 O(log⁡n)O(\log n)O(logn) 的数据结构。它满足如下性质：

1.左子树所有节点的值都小于当前节点的值。

2.右子树所有节点的值都大于当前节点的值。

BST的所有操作都由此引出。在理想情况下，插入、删除等操作正比于树的深度，因此理想复杂度为 O(n)O(n)O(n)。
